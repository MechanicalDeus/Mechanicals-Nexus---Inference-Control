# Usage metrics figures

Committed files here are **SVG placeholders** so `docs/usage-metrics.md` and the README do not show broken image icons on GitHub.

To replace a placeholder with a **real Cursor usage / billing screenshot**, export a PNG from Cursor, name it exactly as in the placeholder subtitle (e.g. `without-nexus-1.png`), place it in this directory, update the matching `![…](…)` link in the docs from `.svg` to `.png`, and remove the `.svg` if you no longer need it.

Original informal filenames (local-only) are listed in [`docs/usage-metrics.md`](../../usage-metrics.md).
